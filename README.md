# AI Logic Puzzle Solver - Futuristic Design

A modern web application for solving Sudoku and Kakuro puzzles using advanced Constraint Satisfaction Problem (CSP) techniques.

## Features

- **Futuristic Cyber Design**: Neon colors, glowing effects, animated backgrounds
- **Sudoku Solver**: 9×9 grid with difficulty levels (Easy, Medium, Hard, Expert)
- **Kakuro Solver**: Customizable grid sizes (4×4, 7×7, 10×10)
- **AI Algorithms**: Backtracking, AC-3, MRV heuristics, Forward Checking
- **Real-time Visualization**: Watch the AI solve puzzles step-by-step
- **Statistics Tracking**: Time, nodes explored, success rate

## How to Use

1. Open `index.html` in a modern web browser
2. Select either Sudoku or Kakuro
3. Generate a puzzle or input your own
4. Click "Solve with AI" to watch the algorithm work
5. View statistics and step through the solution

## Deployment

Simply host the files on any web server or open `index.html` locally.
